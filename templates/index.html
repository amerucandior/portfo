Simon Game - Neon Arcade Edition I architected a fully interactive web game using JavaScript (ES6+) and jQuery to handle game state and DOM manipulation. The project reimagines the classic 80s "Simon" game with a modern "Glassmorphism" aesthetic.
Key Features:

  • State Management: Implemented logic to compare user input arrays against randomly generated game patterns in real-time.
  • Data Persistence: Integrated the Browser localStorage API to save and retrieve high scores across sessions.
  • Responsive Design: Optimized for both desktop (keyboard/mouse) and mobile (touch events) using CSS Grid and Flexbox.
  • Audio/Visual Feedback: synchronized audio cues with CSS animations for an immersive arcade feel.
